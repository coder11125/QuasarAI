# ☄️ Quasar AI

A sleek, modern, multi-provider AI chat interface that runs entirely in your browser. Connect to Google Gemini, OpenAI, Anthropic Claude, Groq, or OpenRouter—all from one beautiful, privacy-focused interface.

## ✨ Features

- **🤖 Multi-Provider Support** — Seamlessly switch between leading AI providers:
  - Google Gemini (3 Flash, 3.1 Pro, 2.5 Pro, etc.)
  - OpenAI (GPT-4, GPT-3.5, etc.)
  - Anthropic Claude (4.6 Sonnet, 4.5 Sonnet, 4.5 Haiku, 4.6 Opus etc.)
  - Groq (Ultra-fast inference)
  - OpenRouter (Access to multiple models)

- **🔒 100% Privacy-First** — All data stays local:
  - API keys stored in browser localStorage
  - Chat history never leaves your device
  - Direct API connections—no proxy servers

- **💬 Rich Chat Experience**:
  - Full Markdown rendering with syntax highlighting
  - Image attachments and vision capabilities
  - Voice input (Web Speech API)
  - Persistent chat history with search
  - Auto-resizing text input

- **🎨 Beautiful UI**:
  - Glassmorphism design with gradient backgrounds
  - Light/Dark mode with system preference detection
  - Responsive sidebar with collapsible navigation
  - Smooth animations and transitions
  - Mobile-friendly responsive layout

- **⚡ Developer-Friendly**:
  - Zero backend required—pure client-side
  - Easy to deploy (GitHub Pages, Netlify, Vercel, etc.)
  - Simple configuration via settings UI
